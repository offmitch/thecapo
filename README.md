# The Capo

The Capo is a web app that recommends songs based on a user’s input. Enter a song or a mood, and The Capo will suggest a track that fits your vibe.  

## Features

- Input a song or mood to get song recommendations  
- Visually engaging interface using **Three.js**  
- Work in progress — new features and improvements coming soon  

## Tech Stack

- Java  
- HTML, CSS, JavaScript  
- Three.js  

## Notes

- The project is currently in development, so some features may be incomplete  
- Future updates will include additional recommendation logic, UI enhancements, and more customization options
